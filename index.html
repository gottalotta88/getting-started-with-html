HTML notes HTML: Hypertext markup language

-Web invented in the 1990s; all pages link together
-Browser makes requests to the internet via HTTP to servers, which respond with files that the browser displays

-Text content: infromation you see
-Markup: what it looks like
-References to other documents: images and videos
-Test out code at link
Opening tags and closing tags; both tags plus contents within is called an element
  • B tag is for bold
  • em tag is for italics
  • If you don’t close tag; everything after is bolded/italicized
  • a is for anchor
    • Used for making links
    • Followed by an attribute name, value, and then contents
  • img is for image
    • Src and alt are attributes; alt is what to display if broken request
    • Img doesn’t have any content, so no closing tag; this is called a void tag, a tag that has no content
  • Lines
    • br is a line break to make text go onto next line; also a void tag
    • p is for paragraph; need closing tag after content
    • Br just puts to next line whereas p tag makes an invisible box; br is an inline element while p is a block element; latter creates blocks on the screen
    • Spaces and returns will not be registered
  • Span and div
    • Span can be used to give CSS styles to contents; inline element
    • Div does the same but as a block element

    Thus basic HTML vocab consists of tags, elements, and attributes
